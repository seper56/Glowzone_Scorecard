# GlowZone Scorecard

## Overview
GlowZone Scorecard is a custom web application developed by SEPER DIGITAL for GlowZone 360. It digitizes the company's golf scorecard process into an easy-to-use mobile-friendly web application.

## Status
Completed

## Purpose
Replace paper scorecards with a fast, responsive digital experience that improves usability for customers and staff.

## Client
GlowZone 360
