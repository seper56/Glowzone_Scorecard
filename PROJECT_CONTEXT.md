# GlowZone Scorecard — Project Context

## Project Type
Client project for GlowZone 360.

## Purpose
This project recreates a digital version of the GlowZone 360 scorecard so customers can enter player names and scores, view totals, and share the final scorecard.

## Technology
- HTML
- CSS
- JavaScript

## Main Files
- `index.html` — main page structure
- `styles.css` — layout and visual styling
- `script.js` — scorecard logic and interactions

## Key Features
- Player name input
- Score entry
- Front 9 total calculation
- Mobile-friendly layout
- Shareable scorecard view
- Promotional/ad area
- Branding using GlowZone assets

## SEPER DIGITAL Notes
This was one of SEPER DIGITAL's first real client projects.

Important lessons:
- Match the client's existing visual style closely.
- Mobile layout matters a lot.
- Client revisions should be tracked clearly.
- Reusable scorecard logic could be useful for future entertainment or sports-related projects.
